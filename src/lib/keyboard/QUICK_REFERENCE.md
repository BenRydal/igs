# IGS Keyboard Shortcuts - Quick Reference

## Playback

```
Space        Play/Pause
←            Seek -5s
→            Seek +5s
Home         Jump to start
End          Jump to end
[            Set left marker
]            Set right marker
-            Slow down
+ or =       Speed up
```

## View

```
d            Toggle 2D/3D
r            Rotate right
Shift+R      Rotate left
v            Toggle video
```

## Selection

```
1            Circle mode
2            Slice mode
3            Highlight mode
Esc          Clear all
```

## Display

```
m            Toggle movement
s            Toggle stops
a            Toggle alignment
c            Toggle colors
```

## Application

```
Shift+?            Show shortcuts
h                  Help modal
o                  Open file
Ctrl/Cmd+Shift+S   Download codes
Ctrl/Cmd+K         Command palette
```
